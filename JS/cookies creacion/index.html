Exemple: Cookie

En el següent document web es designen quatre paràmetres de nom, valor, expiració i parts de ruta enviades a la funció 'CookieSet'. Les parts secure i domini no són definides. Si caduca rebrà valor buit, establirà la data de venciment a 9 mesos a partir de la data actual i si la ruta rep un valor buit, el directori actual i els subdirectoris seran la ruta d'accés. toUTCString converteix la data en una cadena, utilitzant la convenció de temps universal.

	function CookieSet (cName, cValue, cPath, cExpires) {
		cvalue = encodeURIComponent(cValue);
		if (cExpires == ""){
			var cdate = new Date();
			cdate.setMonth(cdate.getMonth() + 9);
			cExpires = cdate.toUTCString();
		}
		if (cPath != ""){
				cPath = ";Path=" + cPath;
		}
		document.cookie = cName + "=" + cValue +"expires=" + cExpires + cPath;
	}

	CookieSet("Name","George ","/","");
	console.log('La cookie és:' + document.cookie);
